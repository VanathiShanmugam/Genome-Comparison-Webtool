# Genome Comparison & Analysis Tool

### Accessible Bioinformatics Web Application with Speech Assistance

---

## Highlights

* Genome comparison with mutation detection
* Visual nucleotide analysis (charts & graphs)
* DNA → Protein translation (6-frame analysis)
* Detailed sequence insights (GC content, motifs, codon usage)
* Speech-assisted output for accessibility
* High-contrast mode & font controls (blind-friendly design)

---

## Overview

This project is an **interactive web-based genome analysis platform** designed to make bioinformatics tools accessible and user-friendly.

It enables users to:

* Compare DNA sequences
* Analyze nucleotide composition
* Generate biological insights
* Translate DNA into protein sequences

Special emphasis is placed on **accessibility**, including speech output and visual adaptability.

---

## Key Features

### Genome Comparison

* Detects mutations between two sequences
* Alignment visualization
* Voice-based report reading

---

### Visual Analysis

* Nucleotide frequency charts
* GC/AT content analysis
* Interactive visualization

---

### Detailed Insights

* Motif detection
* Reverse complement generation
* Codon usage statistics

---

### DNA → Protein Translation

* Six-frame translation
* Start codon validation
* Downloadable outputs
